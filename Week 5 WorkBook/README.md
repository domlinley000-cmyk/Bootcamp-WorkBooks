# Week 5 Summary
In Week 5, I developed my understanding of data protection laws, cloud computing concepts, and Microsoft Azure. I explored how organisations can securely manage customer and business data while transitioning from manual systems to cloud-based solutions.

### Key Learnings & Projects

#### Cloud Computing Fundamentals
I researched what cloud computing does in real-world environments and how it benefits businesses. I compared cloud computing with traditional on-premises infrastructure and explained advantages such as scalability, cost efficiency, remote access, and automated backups.
I also researched and compared major cloud providers, including:
* Microsoft Azure
* Amazon Web Services
* Google Cloud Platform
This included outlining their features, functions, and typical business use cases in a comparison table.

#### Cloud Models & Service Offerings
I researched and explained different cloud service models:
* IaaS (Infrastructure as a Service)
* PaaS (Platform as a Service)
* SaaS (Software as a Service)
I also explored cloud deployment models, including public, private, hybrid, and community cloud, and explained how each could be used in a business environment.

#### Data Protection & Legal Compliance
A key focus was understanding UK data laws and cybersecurity legislation. I researched and applied:
* The General Data Protection Regulation
* The Data Protection Act 2018
* The Computer Misuse Act 1990
* The Police and Justice Act 2006
This helped me understand how organisations must lawfully collect, store, and protect personal data, particularly customer information.

#### Azure Labs & Certification Practice
I completed hands-on labs in Azure exploring:
* Relational and non-relational data
* Data analytics services
* Cloud storage solutions
* Identity and access management
I also completed the DP-900 (Azure Data Fundamentals) practice exam and achieved a score of 80%, demonstrating my understanding of core Azure data concepts.

#### 1500-Word Case Study: “Paws & Whiskers”
The main assessment was a structured 1500-word report analysing how a small pet shop could transition from spreadsheets to Azure. Within the report, I:
* Identified relevant data protection laws affecting customer data
* Recommended Azure services for data storage and analysis
* Designed a relational database model including tables, primary keys, and relationships
* Suggested appropriate data formats (CSV, JSON, Parquet)
* Proposed security measures, backup strategies, and disaster recovery solutions
* Recommended Power BI integration for reporting and data visualisation
